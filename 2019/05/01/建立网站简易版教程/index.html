建立网站简易版教程

2019-05-01

1.去nodejs官网下载安装包（记得选择add to path选项），安装好后打开Node.js command prompt, 输入
node -v
npm -v

显示如下（版本可不同)

2.安装hexo, 输入
npm install -g hexo-cli

注意要将hexo添加到path。

编辑用户环境变量为：
C:\Users\你的计算机名字\AppData\Roaming\npm

3.打开cmd, 建立一个专用文件夹如Blog，进入其中，

hexo init

npm install(第一次使用hexo必须运行这个)

hexo g(hexo generate:生成public文件，html)

hexo s(hexo server)

hexo d(hexo deploy)

注意：1. hexo clean

2.删除deploy.git去掉无.git的影响
